# Lógica de Programação na Prática — Repositório Companion

**Autor:** Kelvin Biffi  
**Livro:** Lógica de Programação na Prática: Aprenda a Pensar Como Programador e Escreva Seu Primeiro Código em Python

Este repositório contém todos os exemplos de código e exercícios resolvidos do livro.

## Como usar

Cada pasta corresponde a um capítulo do livro.  
Você não precisa instalar nada além do Python 3 para rodar os exemplos.

```
python cap-05-sequencia/troco.py
```
